# Generate CEO Briefing - Gold Tier

Generate comprehensive weekly business briefing from all data sources.

---

## When to Use

Use this skill when:
- Scheduled Monday 10 AM briefing
- User requests weekly report
- Business performance review needed
- Investor/stakeholder update needed
- Monthly/quarterly summary

---

## Instructions

### Step 1: Determine Report Period

Calculate the period:
- **Default:** Last 7 days (Monday to Sunday)
- **Custom:** User-specified date range

### Step 2: Fetch Data from All Sources

#### Odoo (Financial Data)
```
[SKILL] Using skill: /get-financial-report
[MCP] Calling mcp__ai-employee__get_financial_report(period="week")
```

#### Facebook (Social Media)
```
[MCP] Calling mcp__ai-employee__get_facebook_insights()
[MCP] Calling mcp__ai-employee__get_posts(limit=10)
```

#### LinkedIn (Social Media)
```
[MCP] Calling mcp__ai-employee__get_linkedin_profile()
```

#### Gmail (Email Metrics)
Count emails sent/received from /Done/email/ folder

#### WhatsApp (Message Metrics)
Count messages sent/received from /Done/whatsapp/ folder

#### Vault (Task Metrics)
Count files in /Done/ by category

### Step 3: Generate Briefing Content

Create comprehensive briefing:

```markdown

---
```

### Step 4: Write Briefing File

Save to vault:
```

```

### Step 5: Log the Action

```

[DONE] Briefing generated: 2026-03-09_Monday_Briefing.md
```

## Data Sources Summary

| Source | Data Fetched | MCP Tool |
|--------|--------------|----------|
| **Odoo** | Revenue, expenses, profit, invoices | `get_financial_report()` |
| **Facebook** | Posts, engagement, reach | `get_facebook_insights()`, `get_posts()` |
| **LinkedIn** | Posts, profile updates | `get_linkedin_profile()` |
| **Gmail** | Email count, response time | Folder scan |
| **WhatsApp** | Message count, response time | Folder scan |
| **Vault** | Tasks completed, pending | Folder scan |

## Error Handling

| Error | Action |
|-------|--------|
| Odoo not connected | Note "Financial data unavailable" in briefing |
| Facebook token expired | Note "Facebook metrics unavailable" |
| No data for period | Show zeros, note "No activity in this period" |
| Vault folders missing | Create folders, note in briefing |
